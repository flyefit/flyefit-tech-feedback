# FLYEfit Tech Feedback

This repository is used to collect public feedback, ideas, and suggestions related to FLYEfit digital products.  
This includes the Mobile App, API integrations, in-gym digital systems, kiosk software, and website functionality.

## Purpose
The Discussions section provides a place for members, partners, and developers to:
- Suggest new features
- Share feedback on existing digital functionality
- Discuss improvements to FLYEfitâ€™s technical platforms

## Scope
This space is for **digital and technical feedback only**.

It is not intended for:
- Gym equipment or facility feedback
- Membership or subscription questions
- Class or timetable issues
- General gym operations

## Member Service
For account issues, membership queries, billing questions, or any non-technical support, please contact [**memberservice@flyefit.ie**](mailto:memberservice@flyefit.ie).

## How to Participate
1. Open the **Discussions** tab.
2. Choose a category.
3. Start a new discussion or join an existing one.
